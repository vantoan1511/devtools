# Changelog

All notable changes to this project will be documented in this file.

## [1.1.1] - 2026-04-06

### Fixed
- Build issue with `!` character following tailwind util classes.

## [1.1.0] - 2026-04-06

### Added
- **RegEx Tester**: New tool for real-time regular expression testing with highlighting and group extraction.
- **Comparison Tool**: Side-by-side and inline text difference analysis.
- **Hash Generator**: Support for MD5, SHA-1, SHA-256, and SHA-512.
- **JSON Formatter**: Format, validate, and minify JSON data.
- **Base64 Tool**: Encode/decode strings and files.
- **Progress Tracking**: Added `PROGRESS.md` to track feature roadmap.
- **Glassmorphism UI**: Enhanced UI with frosted-glass effects and smooth transitions.

### Changed
- Improved sidebar navigation with better active state tracking.
- Updated Home dashboard with cards for all new tools.

### Fixed
- RegEx Tester match visibility issues.
- Type-checking errors in OpenAPI Editor.

## [1.0.0] - 2026-03-20
- Initial release with OpenAPI Editor.
