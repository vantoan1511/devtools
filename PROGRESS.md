# DevTool+ Progress Tracking

## Project Overview
A web-based suite of development utilities built with Vue 3, Vite, PrimeVue v4, and Tailwind CSS.

---

## 🛠️ Feature Roadmap

### Core Infrastructure
- [x] Vue 3 + Vite Setup
- [x] PrimeVue v4 Integration
- [x] Tailwind CSS Styling
- [x] Dark/Light Mode Support
- [x] Responsive Sidebar Navigation
- [x] Glass-morphism UI Theme

### Implemented Tools
- [x] **OpenAPI Editor**: Monaco-based editor with Swagger UI preview and profile management (up to 5 profiles).
- [x] **JSON Formatter**: Format, validate, and minify JSON data.
- [x] **Base64 Tool**: Encode/decode strings and files to/from Base64.
- [x] **Hash Generator**: MD5, SHA-1, SHA-256, SHA-512 generator.
- [x] **Comparison Tool**: Side-by-side and inline text difference analysis.
- [x] **RegEx Tester**: Real-time regular expression testing with highlighting and group extraction.
- [x] **About Page**: Project information and credits.

### 🚀 Next Steps (Priority)
- [ ] **URL Encoder/Decoder**: Handle URL encoding and decoding of parameters.
- [ ] **JWT Debugger**: Decode and inspect JSON Web Tokens.
- [ ] **Unit Testing**: Implement tests for core utility logic (Regex, Base64, etc.).
- [ ] **Copy-to-Clipboard Consistency**: Ensure all tool outputs have easy "Copy" buttons.
- [ ] **Export/Import Profiles**: Ability to export/import OpenAPI profiles as a JSON file.

### 📅 Future Ideas
- [ ] **SQL Formatter**: Format and beautify SQL queries.
- [ ] **Cron Expression Generator**: Visual tool for building cron schedules.
- [ ] **Progressive Web App (PWA)**: Add offline support and installability.
- [ ] **HTTP Client**: Simple tool for testing API requests.
